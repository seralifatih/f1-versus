# f1·versus

**f1-versus.com is the GOAT calculator for Formula 1.** Users pick a preset formula ("Stats Geek", "Era Adjusted", "Peak Performance", "Longevity Weighted") or build their own with sliders, and instantly see how all-time F1 drivers rank under their definition of "greatness". The output is a shareable ranking that fuels the eternal "Hamilton vs Schumacher vs Senna vs Verstappen" debate — one tweet at a time.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.
